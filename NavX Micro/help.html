NavX Micro

What is it?

navX-Micro is a 9-axis inertial/magnetic sensor and motion processor. Designed for plug-n-play installation onto robots such as those used in the FIRST Technology Challenge (FTC) and the FIRST Robotics Challenge (FRC), navX-Micro helps build better robots by adding sophisticated navigation capabilities.

Properties

CAN ID
The ID of the CAN device. This must match the ID of the device configured with the roboRIO WebDash.
