LimeLight

What is it?

Limelight is a plug-and-play smart camera purpose-built for FIRST Robotics Competition.

Add to the build.gradle

repositories { maven { url 'https://jitpack.io' } }

dependencies { implementation 'com.github.GraniteCityGearheads3244:Robot_Libraries:v0.0.1-alha.0'}

Properties

Table Name
Leave as the default "limelight" unless you are using multiple LimeLights on the same robot.
